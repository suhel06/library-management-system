# 📚 Library Management System

A simple web-based Library Management System built using HTML, CSS, and JavaScript.

## 🚀 Features

* 🔐 Login System (Password hidden)
* 📚 Add Book (Maintenance Module)
* 📖 Issue Book (with validation & auto return date)
* 🔄 Return Book
* 💰 Fine Payment System
* 👤 Membership Module
* 🔍 Search Books
* ✅ Form Validation

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* LocalStorage (for data handling)

## 📂 Project Structure

* index.html → Login Page
* dashboard.html → Navigation
* addBook.html → Add Books
* issue.html → Issue Books
* return.html → Return Books
* fine.html → Fine Payment
* membership.html → Membership
* script.js → Logic
* style.css → Styling

## 📌 Note

This project uses LocalStorage to simulate database operations and demonstrates the complete workflow of a Library Management System.
